# Run in dev mode
1. Clone the repo
2. Run 
```
yarn install
```
3. Install Nest CLI
```
npm install -g @nestjs/cli
```
4. Lift the database
```
docker-compose up -d
```

# Stack:
* MongoDB
* NestJS
